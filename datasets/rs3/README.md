# RoadSimulator3 — Telemachus Format

Synthetic telematics data generated by RoadSimulator3, in [Telemachus v0.8](https://doi.org/10.5281/zenodo.17228091) format.

## Source

Edet, S. (2025). *RoadSimulator3: Physics-based telematics simulator.*
GitHub: [SebE585/RoadSimulator3](https://github.com/SebE585/RoadSimulator3)

## Contents

| File | Format | Size | Description |
|------|--------|------|-------------|
| `manifest.yaml` | YAML | 1 KB | SPEC-02 dataset manifest |
| `rs3-telemachus.parquet` | Parquet | 9.1 MB | Full dataset (131,186 rows, 1 trip) |
| `rs3-telemachus.csv` | CSV | 26 MB | Same data, human-readable |

## Simulation

Single trip around Le Havre (Normandie, France), ~3.6 hours.
GPS 1 Hz, IMU 10 Hz, with realistic noise model and gravity vector.

## Usage

```python
import telemachus as tele
df = tele.read("rs3-telemachus.parquet")
```
